🔬 Experimental Journey

Phase 1: Baseline
8%

Keyword search using BM25 only. Established the challenge.

Phase 2: Semantic Search
26%

Added sentence-transformers embeddings. 3.25x improvement!

Phase 3: Hybrid + Reranking
62%

Combined semantic + BM25 with cross-encoder reranking.

Phase 4: Ultimate RAG
64%

Query expansion + page-aware voting. Peak performance!

Phases 5-6: Over-Optimization
52-64%

6 attempts failed to improve. Learned when to stop!

💡 Key Insights

✅ What Worked

  • Hybrid search (semantic + BM25)
  • Large chunks (3000 chars)
  • Cross-encoder reranking
  • Query expansion
  • Page-aware voting

❌ What Failed

  • Over-optimization (64% → 52%)
  • Multi-stage retrieval
  • Extreme voting parameters
  • Very large chunks (4000+)

🎓 The Learning

"Simple, well-tuned solutions beat complex ones. After 6 failed optimization attempts, I learned to recognize fundamental limits and when to stop adding complexity."
